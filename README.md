# skills-getting-started-with-github-copilot2
Exercise: Get started using GitHub Copilot

## Overview
This repository contains a Flask web application that demonstrates GitHub Copilot capabilities. The "test2" functionality showcases how AI-powered development can accelerate coding and improve productivity.

## Features
- **Interactive Web Application**: Built with Flask and modern web technologies
- **Test2 Demo Page**: Dedicated page demonstrating Copilot-generated features
- **Responsive Design**: Mobile-friendly interface with modern styling
- **Interactive Elements**: JavaScript functionality for user engagement

## Getting Started

### Prerequisites
- Python 3.7 or higher
- pip (Python package installer)

### Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/josezarzaf/skills-getting-started-with-github-copilot2.git
   cd skills-getting-started-with-github-copilot2
   ```

2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

3. Run the application:
   ```bash
   cd src
   python app.py
   ```

4. Open your browser and visit `http://localhost:5000`
